Pidrila Boi

Release date: 2017-01-14 ~ 2017-04-26

This game was a small project for a random friendly contest for 24-hour games and this was my entry (I won by the way). A silly game about a so-called Pidrila cat who needs to save everyone.

Slightly updated because of the few requests to the version '2.0'. The controls are switched to WASD, has an ending point beyond 9000 score and new world tiles.
