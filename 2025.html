Programming

  • Serious Sam Classics Patch

    • Slowed down the development of the patch for this year, mostly just fixing and improving upon what's already there. Download this year's last 1.9.4 version here.

  • Serious Engine 1 fork

    • Some more major changes, including but not limited to: SDL3 support that replaced SDL2, cross-platform OpenAL support that also restores EAX effects, multi-mice support, integer-based ticks that completely eliminate the slowdown bug. Not working on it anymore for the time being.

  • Entity Class Compiler

    • Ported some features from the enhanced configuration to the other compilers (1.10, 1.50 and SSR). Also made it buildable under Linux.

  • VDF reader and writer

    • Made a public domain reader/writer library for the Valve Data Format as a personal exercise in C. I really like writing my own parsers of text formats for some reason.
