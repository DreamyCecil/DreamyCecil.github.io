Advanced Multiplayer 2 Mod

Advanced Multiplayer 2 is a mod that adds a lot of options for multiplayer games including The First Encounter map support and Cooperative mode support for singleplayer maps.

This mod is currently in a public beta stage.

Dreamy GRO

A custom utility designed to help mappers automatically pack their Serious Sam levels into GRO archives with all the needed resources that levels may use.

Run For Life (Demo)

A mapping jam event hosted by Rakanishu. "Run For Life" map was submitted by me.

Extended Warped Texture Pack

A texture pack with variations of textures from Seriously Warped Deathmatch 3.0 mod.

Heavy Voice Lines

A sound pack that replaces Serious Sam voice lines with the ones by Heavy Weapons Guy from Team Fortress 2.

Custom Serious Sam Patch

An unofficial patch for classic Serious Sam games that includes a lot of modern enhancements that aren't present in the original code, nor in the official Steam patch.

Chaosbox

This is a mod that consists out of many smaller mods and ideas, which when combined together, give you quite a chaotic gameplay experience.

Bright Island Classic

A port of the Bright Island campaign from Serious Sam Revolution back to The Second Encounter.

Remastered Map Collection

A collection of old maps with small refreshing updates to them.

Cecil's Bot Mod

A simple mod that adds Cooperative and Deathmatch player bots.

Designed with other mod integration in mind.

Half-Life 2 Immersion Mod

Half-Life 2 Immersion Mod tries to recreate an atmosphere of playing Half-Life 2 on Serious Engine with ported weapons, enemies, effects and a bunch of other features that mimic Half-Life 2 as closely as possible!

Serious Sam SDK & Example Entities Library

This is the original Serious Sam SDK by Croteam that includes the following projects: EntitiesMP, GameGUIMP, GameMP. There's also a separate SDK for creating your own Serious Sam entity libraries.

The project files were created for newer Visual Studio versions (starting with 2013).
